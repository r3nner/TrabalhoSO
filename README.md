# TrabalhoSOok

Este repositório está organizado em duas entregas:

- `T2/`: versão referente à entrega do trabalho T2.
- `T3/`: versão referente à entrega do trabalho T3 (anteriormente disponível diretamente na branch `master`).

Cada pasta contém seus próprios artefatos (Makefile, fontes, programas montados etc.). O diretório raiz permanece limpo, facilitando a navegação e o empacotamento das entregas.
